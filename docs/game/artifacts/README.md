## Collection Standards

All artifact documentation follows the Historical Artifact Metadata Standard, including:

- **Official Name**: Standardized English name
- **Original Name**: Historical name in original language
- **Civilization**: Sumer, Akkadian, Babylonian, Assyrian, etc.
- **Historical Period**: Period classification
- **Approximate Date**: Date range in BCE/CE
- **Current Museum**: Current repository
- **Discovery Site**: Archaeological find location
- **Category**: Object classification
- **Material**: Physical composition
- **Dimensions**: Size specifications
- **Historical Description**: Detailed description
- **Political Context**: Role in political/administrative systems
- **Historical Importance**: Significance assessment
- **Discovery Story**: Archaeological context
- **Interesting Facts**: Notable details
- **Educational Summary**: Learning points
- **References**: Museum and academic sources
