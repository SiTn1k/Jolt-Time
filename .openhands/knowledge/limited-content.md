# Jolt Time — Limited-Time Content and Legacy Systems

## Overview

Limited-time content creates excitement and milestones while respecting player time and avoiding unhealthy FOMO mechanics. Jolt Time's approach celebrates participation and provides returning players with opportunities to obtain similar rewards through alternate means. All limited-time rewards remain cosmetic-only, ensuring fair gameplay for all players regardless of when they joined.

## 5. Returning Content Philosophy

Jolt Time ensures players who miss limited-time content have opportunities to obtain similar rewards.

### Return Mechanisms

| Mechanism | Description | Examples |
|-----------|-------------|----------|
| **Recolored Variants** | Same item with different colors | "Golden Spring Frame" → "Silver Spring Frame" |
| **Anniversary Editions** | Updated versions for later anniversaries | "2024 Anniversary Frame" → "2025 Anniversary Frame" |
| **Rerun Events** | Popular events return with new missions | Egypt Week returns with new rewards + old available |
| **Heritage Shop** | Past items available for alternate currency | Festival Coins, Anniversary Tokens |
| **Legacy Equivalents** | Similar items with different designs | Similar frame, different theme |

### Recolored Variant Examples

| Original | Variant | How to Obtain |
|----------|---------|---------------|
| Cherry Blossom Frame (Pink) | Moonlight Blossom Frame (Silver) | Heritage Shop (Festival Coins) |
| Golden Sun Frame | Silver Sun Frame | Complete rerun event |
| Frost Crystal Frame | Ember Crystal Frame | Anniversary event |
| Ankh Frame (Gold) | Ankh Frame (Silver) | Legacy collection completion |

### Rerun Event Policy

- Popular events may rerun 12+ months after original
- Rerun events include original rewards available via alternate methods
- New rewards added alongside original content
- Players who completed original still receive new content

## 10. Anti-FOMO Philosophy

Jolt Time respects player time and avoids unhealthy urgency mechanisms.

### Core Principles

#### Respect Player Time

| Principle | Implementation |
|-----------|---------------|
| No Mandatory Daily Login | Events designed for flexible participation |
| No Missed Opportunity Punishment | Tokens convert to dust if unspent |
| Reasonable Time Investment | 5-10 hours total for full event completion |
| Flexible Pacing | Events last 7-30 days, no daily requirements |

#### Avoid Unhealthy Urgency

| Concern | Jolt Time's Approach |
|---------|---------------------|
| Countdown Timers | Informational only, no penalties |
| "Last Chance" Messaging | Rare, factual, never manipulative |
| Exclusive Deadlines | Alternatives always available |
| Limited Quantity Scarcity | Used sparingly for special items only |
| Social Proof Pressure | No "X players have claimed" manipulation |

#### Celebrate Participation

| Instead of... | Jolt Time offers... |
|--------------|---------------------|
| "Only 100 available!" | "Earn this exclusive reward!" |
| "Ends tomorrow!" | "You have 7 more days to participate." |
| "You're missing out!" | "Thanks for joining us!" |
| "Top 10% get reward" | "Everyone who participates earns something." |
| "Never available again!" | "This will return as Heritage in the future." |

### FOMO Prevention Features

| Feature | Specification |
|--------|--------------|
| Event Reruns | Popular events return 12+ months later |
| Heritage Alternatives | Similar items available via alternate methods |
| Flexible Duration | 7-30 day events, no daily mandatory actions |
| Token Conversion | Unspent tokens convert to permanent currency |
| Alternative Rewards | Multiple paths to similar cosmetic outcomes |
| No Exclusive Loss | No permanent exclusive content that affects gameplay |

### Fair Play Promise
