# Jolt Time — Events System

## Overview

Events provide limited-time content that keeps the game fresh and exciting. All events are cosmetic-focused with no pay-to-win rewards.

---

## Event Types Detail

### 1. Collection Events

**Purpose:** Focus on gathering specific artifacts

```
event_type: collection
focus: specific_era_or_set
duration: 7_days
activities:
  - "Collect X fragments"
  - "Open event capsules"
  - "Complete event missions"
rewards:
  - event_capsule
  - cosmetic_items
  - cosmetic_badges
```

event_type: challenge

## Event Balance Rules

### No Pay-to-Win

- ❌ No exclusive power upgrades
- ❌ No gameplay-altering items
- ❌ No energy shortcuts
- ✅ Cosmetics only
- ✅ All obtainable through gameplay
- ✅ No "pay to complete" mechanics

### Fair Acquisition

| Reward Type | F2P Achievable | Time Required |
|-------------|----------------|---------------|
| Basic Badge | Yes | 2-3 hours |
| Event Frame | Yes | 5-10 hours |
| Event Title | Yes | Complete event |
| Event Aura | Yes (rare) | 10+ hours |
| Limited Skin | Yes (very rare) | Event + luck |

### EventCapsule Contents

```
event_capsule:
  cost: Free (event missions) or 100 Chrono Dust
  
  contents:
    - cosmetic_items: 100%
    - fragments: 50%
    - chrono_dust: 50%
    
  no_contents:
    - artifacts
    - premium_currency
    - power_items
```

### Notification Types

| Type | Frequency | Message |
|------|-----------|---------|
| Event Start | Once | "Egypt Week begins! Join now!" |
| Daily Reminder | Max 1/day | "Your event rewards await!" |

| Goal Complete | 100% | "Reward unlocked for all!" |
