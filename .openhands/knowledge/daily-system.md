# Jolt Time — Daily Systems

## Overview

Daily systems are designed to create habitual engagement. Each day should feel rewarding and give players a reason to return.

---

## Return Player Bonuses

### Welcome Back Rewards

| Days Absent | Welcome Reward |
|-------------|---------------|
| 1 day | Normal daily reward |
| 2-3 days | 1.5x daily reward |
| 4-7 days | 2x daily reward + Free Capsule |
| 8-14 days | 2.5x daily reward + Rare Capsule |
| 14+ days | 3x daily reward + Free Legendary Capsule |

### Streak Protection

- **Grace Period:** 24 hours after streak ends
- **Protection Item:** "Temporal Shield" (rare drop) extends grace by 48 hours
- **Recovery:** Complete 3 missions to restore full streak (once per week)

## Technical Implementation

### Reset Timing
- All resets at 00:00 UTC
- Server-side validation
- Client shows local time with UTC reference

### Progress Persistence
- Real-time sync with server
- Offline progress calculated on login
- Quest progress saved immediately

### Notification Strategy
- Non-intrusive reminders
- User controls notification preferences
- Never more than 2 notifications per day

---

## Player Experience Goals

### Make Players Feel
- **Welcomed** — Every login is rewarding
- **Progressing** — Visible daily improvements
- **Learning** — Discovering history naturally
- **Respected** — Time is valued, no forced grinding

### Avoid
- **Punishment** — Missing a day should not feel devastating
- **Overwhelm** — Keep options clear and simple
- **Annoyance** — Notifications remain useful and limited
- **Boredom** — Daily variety prevents monotony

---

*Every day in Jolt Time should feel like a small celebration.*
